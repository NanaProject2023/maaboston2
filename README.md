


Welcome to the Boston Mini Dashboard! This is a lightweight, interactive dashboard focused on providing real-time insights and searches for Boston residents and visitors. While currently simple, the dashboard is designed to scale in the future, potentially integrating high-end real estate and more.
Features
	•	Boston News: Stay up-to-date with the latest news from Boston via integrated APIs.
	•	Weather Component: Get the current weather in Boston, using weather APIs to deliver real-time updates.
	•	Search Functionality: Explore cafés, libraries, and restaurants in Boston, using a powerful search powered by multiple APIs, including Google and YouTube APIs.
	•	What's Happening in Boston: A dynamic news and event overview, so you’re always informed.
	•	Navigation: The Boston logo takes you back to the main page, and a search bar in the navbar lets you quickly look up what you need.
Tech Stack
	•	Frontend: [React.js]
	•	APIs: Google APIs, YouTube APIs, RSS News Feeds, Weather APIs
	•	Parsing: XML through RSS feeds
Future Plans
In upcoming versions, we plan to incorporate more specialized Boston real estate searches and other niche services. Stay tuned!
